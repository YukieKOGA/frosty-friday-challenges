## 課題内容 (例)
1. サンプルデータをSnowflakeにロードする。
2. 簡単なSELECT文でデータを確認する。
3. ロードしたテーブルに対して集計処理を行う。

## 進め方
- `solution.sql` に解答SQLを記述します。
- `explanation.md` には解法や考え方をまとめます。
- `sample-data/` にインポート用のデータファイルを配置できます。
